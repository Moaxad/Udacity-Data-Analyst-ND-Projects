Conclusions¶

Question 1 Conclusion:¶

The highest movies in income were: Avatar, Star Wars: The Force Awakens, Titanic, Jurassic World and Furious 7. However all of these movies according to the plot have similar amount of budget but huge difference in income!

Question 2 Conclusion:¶

We can see a positive correlation of 0.68 which means there's a strong relationship between the budget and revenue, but that's does not necessarily means one variable is a cause for the other.

Question 3 Conclusion:¶

We can see the plot for the most rated movies according to this dataset, but we have a limitation mentioned in the next section.

Limitations:¶

For question 3, the budget and revenue data for the top rated movies is missing so we cannot compare the top rated movies with the votes, we did not want to drop these movies because half of the dataset have budget as 0 (null) but it's hard to drop 50% of the data and lose some important values like the top 10 rated movies.
